# Joplin Batch

> [安装插件](https://joplinapp.org/plugins/plugin/rxliuli.joplin-batch)

处理一些 Joplin 本身不支持的批量操作，兼容 Joplin 桌面端/移动端。

- 清理不再使用的附件资源
- 在所有笔记中搜索和替换
